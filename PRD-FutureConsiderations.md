## 19. Roadmap de Desarrollo (Fases)

### Fase 1: Fundamentos (MVP)

**Objetivo:** Sistema jugable básico

**Features:**
- ✓ Registro e inicio de sesión
- ✓ Creación de piloto (1 carrera: Minero)
- ✓ Sistema de habilidades básico
- ✓ Sistema Vaxav predefinido
- ✓ 1 estación inicial con módulos básicos
- ✓ Nave minera funcional
- ✓ Minería de 3 tipos de asteroides
- ✓ Sistema de ticks
- ✓ Inventario básico
- ✓ Interfaz funcional con Tailwind

### Fase 2: Economía y Fabricación

**Features:**
- Sistema de mercado
- Refinamiento de minerales
- Fabricación de módulos T1
- Construcción de naves básicas
- Sistema de Blueprints

### Fase 3: Combate

**Features:**
- Combate básico PvE
- NPCs hostiles
- Doctrina de combate
- Sistema de clonación
- Recompensas de combate

### Fase 4: Social

**Features:**
- Corporaciones de jugadores
- Chat corporativo
- Sistema de roles
- Diplomacia básica

### Fase 5: Expansión

**Features:**
- 4 carreras completas
- 4 facciones NPC principales
- Misiones y agentes
- Sistema de relaciones
- Combate PvP

### Fase 6: Endgame

**Features:**
- Stargates y sistemas procedurales
- Naves capitales
- Alianzas
- Guerras de facción
- Construcción de estaciones por jugadores
